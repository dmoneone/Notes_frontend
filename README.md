## Notes Frontend

### How to launch Frontend?

1) You should be sure that the backend has been already run. <br/>
2) git clone https://github.com/dmoneone/Notes_frontend or download<br/>
3) npm install<br/>
4) npm start<br/>

## If you want to run e2e test: 

npm run test:cypress
